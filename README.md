# React-Redux app.

App will demonstrate below things.

	-	react-routing
	-	redux-promise-middleware
	-	redux-logger
	-	Api simulation using mock json.



Steps to Run : 

	1.run npm install
	2.run node server
	3.click link to open home page : http://localhost:3000/dist/index.html


NOTE 
run webpack cmd after changes in component
